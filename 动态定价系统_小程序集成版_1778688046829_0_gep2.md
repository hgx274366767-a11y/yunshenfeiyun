## 一、定价模型概述

### 核心公式

```
P_final = P_base × (1 + λ_terrain + λ_demand + λ_time + λ_risk + λ_policy)
```

| 参数 | 含义 |
|------|------|
| P_base | 基础价格（元） |
| λ_terrain | 地形修正因子 |
| λ_demand | 供需修正因子 |
| λ_time | 时效修正因子 |
| λ_risk | 风险修正因子 |
| λ_policy | 政策合规修正因子 |

### 基础价格计算

```
P_base = 距离(km) × 载重系数 × 单价基准
```

| 载重区间 | 系数 |
|---------|------|
| ≤50kg | 1.0 |
| 50-100kg | 1.2 |
| 100-200kg | 1.5 |
| >200kg | 2.0 |

**单价基准**：3.5 元/km（川渝山区首年定价）

## 二、五维修正因子体系

### 1. 地形修正因子 (λ_terrain)

```javascript
const terrainFactor = {
  PLAIN: 0,           // 平原/城市
  HILL: 0.10,        // 丘陵
  MOUNTAIN: 0.15,    // 山区
  VALLEY: 0.12,      // 峡谷
  CROSSING: 0.15,    // 跨江/跨峡谷
  HIGH_VOLTAGE: 0.12 // 高压线附近
};
```

**叠加规则**：山区 + 跨江 = 0.15 + 0.15 = 0.30

### 2. 供需修正因子 (λ_demand)

```javascript
const demandFactor = {
  BALANCED: 0,           // 供需平衡
  DEMAND_HIGH: 0.15,     // 需求紧张
  DEMAND_LOW: -0.10,     // 需求不足
  HOLIDAY: 0.10,        // 节假日
  MORNING_PEAK: 0.05,   // 早高峰
  EVENING_PEAK: 0.05    // 晚高峰
};
```

**供需比计算**：
```
供需比 = 当前订单数 / 可用飞手数
if 供需比 > 1.5: λ_demand = +0.15
if 供需比 < 0.5: λ_demand = -0.10
```

### 3. 时效修正因子 (λ_time)

```javascript
const timeFactor = {
  STANDARD: 0,        // 标准时效
  URGENT_2H: 0.30,  // 2小时内
  URGENT_4H: 0.20,  // 4小时内
  SAME_DAY: 0.10,    // 当日达
  NIGHT: 0.10,       // 夜间(20:00-06:00)
  EARLY_MORNING: 0.08 // 凌晨(06:00-08:00)
};
```

### 4. 风险修正因子 (λ_risk)

```javascript
const riskFactor = {
  NORMAL: 0,           // 常规作业
  DENSE_POPULATION: 0.08, // 人群密集区
  PRECISION: 0.15,     // 精密吊装
  WIND_LEVEL_4_5: 0.10,  // 中风
  WIND_LEVEL_6: 0.20,     // 大风(不作业)
  LIGHT_RAIN: 0.10,    // 轻雨
  HEAVY_RAIN: 0.30,   // 大雨
  THUNDER: 999,        // 雷电(禁止作业)
  HIGH_TEMP: 0.05,     // 高温>35°C
  LOW_TEMP: 0.08       // 低温<0°C
};
```

### 5. 政策合规修正因子 (λ_policy)

```javascript
const policyFactor = {
```

```typescript
      risk: { factor: number; amount: number };
      policy: { factor: number; amount: number };

  };
}
```

## 四、核心算法实现

### 4.1 价格计算核心逻辑

```typescript
class PricingEngine {
  
  // 单价基准 (元/km)
  private readonly UNIT_PRICE = 3.5;
  
  // 载重系数表
  private readonly weightCoefficients = [
    { max: 50, coefficient: 1.0 },
    { max: 100, coefficient: 1.2 },
    { max: 200, coefficient: 1.5 },
    { max: Infinity, coefficient: 2.0 }
  ];
  
  /**
   * 计算基础价格
   */
  calculateBasePrice(distance: number, weight: number): number {
    const coefficient = this.getWeightCoefficient(weight);
    return distance * this.UNIT_PRICE * coefficient;
  }
  
  /**
   * 计算地形修正因子
   */
  calculateTerrainFactor(terrain: string, isCrossing: boolean): number {
    const factors = {
      PLAIN: 0,
      HILL: 0.10,
      MOUNTAIN: 0.15,
      VALLEY: 0.12,
      CROSSING: 0.15
    };
    
    let factor = factors[terrain] || 0;
    if (isCrossing) factor += 0.15;
    
    return factor;
  }
  
  /**
   * 计算供需修正因子
   */
  calculateDemandFactor(orderCount: number, pilotCount: number): number {
    if (pilotCount === 0) return 0;
    
    const ratio = orderCount / pilotCount;
    
    if (ratio > 1.5) return 0.15;      // 需求紧张
    if (ratio > 1.2) return 0.10;
    if (ratio < 0.5) return -0.10;     // 供给过剩
    if (ratio < 0.8) return -0.05;
    
    return 0; // 供需平衡
  }
  
  /**
   * 计算时效修正因子
   */
  calculateTimeFactor(
    timeRequired: string, 
    isNight: boolean,
    currentHour: number
  ): number {
    let factor = 0;
    
    // 时效加价
    switch (timeRequired) {
      case 'URGENT_2H': factor += 0.30; break;
      case 'URGENT_4H': factor += 0.20; break;
      case 'SAME_DAY': factor += 0.10; break;
    }
    
    // 时段加价
    if (isNight || (currentHour >= 20 || currentHour < 6)) {
      factor += 0.10;
    } else if (currentHour >= 6 && currentHour < 8) {
      factor += 0.08;
    }
    
    return factor;
  }
  
  /**
   * 计算风险修正因子
   */
  calculateRiskFactor(params: RiskParams): number {
    let factor = 0;
    
    if (params.windLevel >= 6) return 999; // 禁止作业
    if (params.windLevel >= 4) factor += 0.10;
    
    if (params.rainLevel === 'HEAVY') factor += 0.30;
    if (params.rainLevel === 'LIGHT') factor += 0.10;
    
    if (params.hasThunder) return 999; // 禁止作业
    
    if (params.temperature > 35) factor += 0.05;
    if (params.temperature < 0) factor += 0.08;
    
    if (params.isPrecision) factor += 0.15;
    if (params.isDensePopulation) factor += 0.08;
    
    return factor;
  }
  
  /**
   * 计算政策合规修正因子
   */
  calculatePolicyFactor(airspaceType: string): number {
    const factors = {
      REGULAR: 0,
      TEMP: 0.05,
      CONTROLLED: 0.10,
      SENSITIVE: 0.15
    };
    
    return factors[airspaceType] || 0;
  }
  
  /**
   * 主计算函数
   */
  calculatePrice(request: PriceCalculateRequest): PriceCalculateResponse {
    // 1. 基础价格
    const basePrice = this.calculateBasePrice(request.distance, request.weight);
    
    // 2. 各维度因子
    const terrainFactor = this.calculateTerrainFactor(
      request.terrainType, 
      request.isCrossing || false
    );
    
    const demandFactor = this.calculateDemandFactor(
      request.currentOrderCount || 0,
      request.availablePilotCount || 1
    );
    
    const timeFactor = this.calculateTimeFactor(
      request.timeRequired || 'STANDARD',
      request.isNight || false,
      new Date().getHours()
    );
    
    const riskFactor = this.calculateRiskFactor(request.riskParams || {});
    
    const policyFactor = this.calculatePolicyFactor(
      request.airspaceType || 'REGULAR'
    );
    
    // 3. 检查禁止条件
    if (riskFactor >= 999) {
      return {
        success: false,
        error: '当前条件下无法作业（风力过大或存在雷电）'
      };
    }
    
    // 4. 合成总修正系数
    const totalFactor = terrainFactor + demandFactor + timeFactor + riskFactor + policyFactor;
    
    // 5. 计算最终价格
    const finalPrice = Math.round(basePrice * (1 + totalFactor) * 100) / 100;
    
    // 6. 价格校验（上下限保护）
    const minPrice = basePrice * 0.7;  // 最低7折
    const maxPrice = basePrice * 3.0;   // 最高3倍
    
    const adjustedPrice = Math.max(minPrice, Math.min(maxPrice, finalPrice));
    
    return {
      success: true,
      data: {
        basePrice,
        finalPrice: adjustedPrice,
        breakdown: {
          terrain: { factor: terrainFactor, amount: basePrice * terrainFactor },
          demand: { factor: demandFactor, amount: basePrice * demandFactor },
          time: { factor: timeFactor, amount: basePrice * timeFactor },
          risk: { factor: riskFactor, amount: basePrice * riskFactor },
          policy: { factor: policyFactor, amount: basePrice * policyFactor }
        },
        validUntil: this.getValidUntil(),
        margin: this.estimateMargin(adjustedPrice, basePrice)
      }
    };
  }
  
  /**
   * 获取价格有效期（15分钟）
   */
  private getValidUntil(): string {
    const expiry = new Date(Date.now() + 15 * 60 * 1000);
    return expiry.toISOString();
  }
  
  /**
   * 估算毛利率
   */
  private estimateMargin(finalPrice: number, basePrice: number): number {
    // 平台佣金比例 10-15%
    const platformCommission = 0.12;
    const platformRevenue = finalPrice * platformCommission;
    const grossProfit = platformRevenue - (finalPrice * 0.03); // 保险成本约3%
    
    return Math.round((grossProfit / finalPrice) * 100 * 100) / 100;
  }
  
  /**
   * 获取载重系数
   */
  private getWeightCoefficient(weight: number): number {
    for (const tier of this.weightCoefficients) {
      if (weight <= tier.max) {
        return tier.coefficient;
      }
    }
    return 2.0;
  }
}
```

### 4.2 供需动态调节

```typescript
class DemandAdjuster {
  
  /**
   * 供需失衡时建议调整策略
   */
  suggestAdjustment(orderCount: number, pilotCount: number): AdjustmentSuggestion {
    if (pilotCount === 0) {
      return { action: 'WAIT', reason: '无可用飞手' };
    }
    
    const ratio = orderCount / pilotCount;
    
    if (ratio > 1.5) {
      // 需求紧张 → 建议涨价吸引飞手
      return {
        action: 'INCREASE_PRICE',
        suggestedAdjustment: 0.15,
        reason: `供需比${ratio.toFixed(2)}，建议上调15%吸引周边飞手`
      };
    }
    
    if (ratio < 0.5) {
      // 供给过剩 → 建议降价激活订单
      return {
        action: 'DECREASE_PRICE',
        suggestedAdjustment: -0.10,
        reason: `供需比${ratio.toFixed(2)}，建议下调10%激活订单`
      };
    }
    
    return {
      action: 'KEEP_PRICE',
      suggestedAdjustment: 0,
      reason: '供需平衡，维持当前价格'
    };
  }
}
```

## 五、计算示例

### 示例1：普通山区订单

```
输入：
- 距离：5km
- 载重：80kg
- 地形：山区
- 时效：当日达
- 时段：白天

计算：
1. 基础价格 = 5 × 3.5 × 1.2 = 21元
2. 地形因子 = 0.15
3. 供需因子 = 0（假设平衡）
4. 时效因子 = 0.10
5. 风险因子 = 0
6. 政策因子 = 0

最终价格 = 21 × (1 + 0.15 + 0.10) = 21 × 1.25 = 26.25元
```

### 示例2：紧急跨江夜间订单

```
输入：
- 距离：3km
- 载重：50kg
- 地形：山区 + 跨江
- 时效：2小时紧急
- 时段：夜间
- 天气：中风

计算：
1. 基础价格 = 3 × 3.5 × 1.0 = 10.5元
2. 地形因子 = 0.15 + 0.15 = 0.30
3. 供需因子 = 0
4. 时效因子 = 0.30 + 0.10 = 0.40
5. 风险因子 = 0.10（中风）
6. 政策因子 = 0

最终价格 = 10.5 × (1 + 0.30 + 0.40 + 0.10) = 10.5 × 1.80 = 18.9元
```

## 九、注意事项

1. **安全优先级**：雷电/大风天气直接返回禁止作业，不进行价格计算
2. **价格保护**：最终价格不超过基础价格 300%，不低于 70%
3. **动态刷新**：价格有效期 15 分钟，超时需重新计算
4. **数据存储**：每次计算记录到 price_history 表，用于后续分析优化
5. **日志记录**：供需比、调整建议等关键决策点需记录日志
