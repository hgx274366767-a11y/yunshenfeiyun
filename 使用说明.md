## 一、项目架构总览

```
云深飞运/
├── apps/
│   ├── client/                  # 客户端小程序 (UniApp Vue3)
│   └── pilot/                   # 飞手端小程序 (UniApp Vue3)
├── supabase/
│   ├── migrations/              # 数据库迁移 (2个文件)
│   ├── functions/               # Edge Functions (16个函数)
│   └── config.toml              # Supabase 项目配置
├── tests/                       # 单元测试 (Vitest, 263 测试)
├── .env.example                 # 环境变量模板
└── CLAUDE.md                    # 项目中枢文档
```

**技术栈：** 前端 UniApp Vue3 + Vite / 后端 Supabase Edge Functions (Deno) / 地图 腾讯位置服务 / 支付 微信 V3 / 导航 高德地图

## 三、环境变量配置

### 3.1 复制模板

```bash
cd 云深飞运/
cp .env.example .env
```

### 3.2 填写实际值

编辑 `.env`，将以下占位符替换为真实值：

```bash
# ========== Supabase (火山引擎) ==========
SUPABASE_URL=https://your-project.volcengineapi.com
SUPABASE_ANON_KEY=eyJ...你的匿名密钥
SUPABASE_SERVICE_ROLE_KEY=eyJ...你的service_role密钥

# ========== 微信客户端小程序 ==========
WX_CLIENT_APPID=wx1234567890abcdef    # 客户端小程序 AppID
WX_APP_SECRET=abc123...               # 小程序 AppSecret
# 注意：这个密钥在整个项目中统一用 WX_APP_SECRET，客户端和飞手端共用

# ========== 微信飞手端小程序 ==========
WX_PILOT_APPID=wx0987654321fedcba    # 飞手端小程序 AppID

# ========== 微信支付 V3 ==========
WX_PAY_MCHID=1234567890               # 商户号
WX_PAY_V3_SECRET=abc123...            # API V3 密钥

# ========== 腾讯位置服务 ==========
QQMAP_KEY=abc123...                   # 腾讯地图 Key

# ========== 高德地图 ==========
AMAP_KEY=abc123...                    # 高德地图 Key
```

### 3.3 同步到 Supabase 云端

Edge Functions 在 Deno 环境运行，无法读取本地 `.env`。需要通过 Supabase 命令行或 Web 控制台为每个需要密钥的函数设置云端 Secret：

```bash
# 逐个设置（推荐在 Supabase Dashboard → Edge Functions → 函数名 → Settings 中操作）
# 或使用 Supabase CLI:
supabase secrets set SUPABASE_URL=https://... --project-ref rhfruguxtprgdlrlmzis
supabase secrets set SUPABASE_SERVICE_ROLE_KEY=eyJ...
supabase secrets set WX_CLIENT_APPID=wx...
supabase secrets set WX_PILOT_APPID=wx...
supabase secrets set WX_APP_SECRET=...
supabase secrets set WX_PAY_MCHID=...
supabase secrets set WX_PAY_V3_SECRET=...
supabase secrets set QQMAP_KEY=...
supabase secrets set AMAP_KEY=...
```

需要设置 Secret 的函数清单（按需）：

| 函数目录 | 需要的 Secret |
|---------|-------------|
| `_shared/` | `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY` |
| `auth/` | 上述 + `WX_CLIENT_APPID`, `WX_PILOT_APPID`, `WX_APP_SECRET` |
| `payment/` | 上述 + `WX_PAY_MCHID`, `WX_PAY_V3_SECRET`, `WX_CLIENT_APPID` |
| `pricing/` | 上述 + `QQMAP_KEY` |
| `notification/` | 上述 + `WX_APP_SECRET`, `SMS_API_ENDPOINT`（可选）, `SMS_API_KEY`（可选） |

在 Supabase Dashboard → Edge Functions → 对应函数 → Settings 中：
- 需要公开访问的函数（被小程序调用的）必须设置 `is_public: true`
- 仅内部调用的函数（如 `credit-calc`）使用 `SERVICE_ROLE_KEY` 鉴权

## 十一、H5 构建已知限制

H5 构建 (`pnpm build:h5`) 依赖 `@dcloudio/uni-h5@3.0.0-alpha` + `vite@5.2.8`：

- **Vant Weapp 组件在 H5 不可用**：`van-*` 组件依赖微信小程序 `Behavior()` API，H5 模式渲染受限。仅登录页（使用原生元素）完整渲染
- H5 构建主要用于开发调试验证和 Playwright E2E 测试
- 生产环境主要目标为微信小程序
